# Comandos úteis do GitHub



## Inicia um novo repositório

> git init

## Clona um repositório

> git clone

## Mostra o estado atual do Git

> git status

## Adiciona um arquivo ou pasta para o Git

> git add meu_arquivo.java

## Commita todos os arquivos adicionados

> git commit -m "mensagem do commit"

## Envia as alterações locais para o remoto

> git push

## Buscar e baixar atualizações do repositório remoto

> git pull
